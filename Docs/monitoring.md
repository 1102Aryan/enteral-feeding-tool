Targets: **fasting/pre-feed 6–10 mmol/L, feeding 6–12 mmol/L** (JBDS §8.1; UHL §2.3). Hyperglycaemia is defined as CBG >12 (JBDS §11).
 
| Band | CBG (mmol/L) | Action | Source |
|---|---|---|---|
| Hypoglycaemia | <4 | Treat per Table 3: 15–20 g rapid CHO via tube (Lift preferred), flush; recheck q15min, repeat up to 3×; IV glucose if persists. Never omit basal (T1). Review insulin. Then ≥4-hourly for 24–48 h. | UHL §2.9.2, Table 3; JBDS §12 |
| Looming hypo | 4–6 | Consider reducing insulin; increase monitoring frequency. | UHL Fig 1 |
| Target | 6–12 | Continue; review at 36–48 h; seek early review if feed regimen altered. | JBDS §8.1; UHL §2.8 |
| Above target | >12 | **Test ketones** (→ pathway below). If no ketones: PRN rapid-acting correction + titrate prior feed insulin **+2–4 units or 10–20%**. Caution within 4 h of a feed break. Persistently >12 on 2 consecutive readings → inform DIT. | UHL §2.9.3, Fig 1; JBDS §11 |
