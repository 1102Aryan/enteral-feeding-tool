### Group A — One-time setup (at feed commencement; DIT/doctor-led, nurse confirms)
- **Diabetes type:** Type 1 / Type 2 / Type 3c–secondary / known insulin deficiency
- **On insulin pump (CSII)?** yes/no
- **Usual basal insulin:** name + dose + usual administration time
- **On metformin pre-admission?** yes/no · **metformin contraindicated?** (renal/AKI/lactic-acidosis risk)
- **Weight (kg)** and **most recent HbA1c** (inform insulin choice — JBDS §6)
- **High-risk flags** (multi-select; UHL Table 1 / JBDS §2.1): T1DM, known insulin deficiency, high HbA1c, high pre-feed glucose variability, hypo in prior 24 h, high NEWS, high tube-removal risk, on steroids
- **Pancreatic exocrine insufficiency / enzymes prescribed?** (Type 3c)
### Group B — Feed & insulin regimen (dietitian/prescriber-led; nurse references)
- **Feed regimen type:** single feed + break (e.g. 20 h + 4 h) / continuous 24 h / multiple short (intermittent) / bolus
- **Feed product**, **infusion rate (ml/hr)**, **total carbohydrate (g)** and **carb per hour (g/hr)** (JBDS §3.3, §8 — carb content drives dosing)
- **Feed duration (hr/day)**, **start time**, **break window** (start–end)
- **Feed-related insulin:** type (rapid analogue / soluble / isophane / premixed / long-acting / none), dose(s) and **timing relative to feed** (start, +4h, +6h, +10h/midpoint…) — JBDS §6.2 stresses timing must be relative to the feed, never a clock time
- **PRN correction insulin:** brand (same as feed bolus — JBDS §6.2) + scale per local policy
- **On VRIII?** yes/no + rate (forces hourly CBG)
### Group C — The repeated bedside loop (the screen nurses live in)
- **CBG value (mmol/L)** + timestamp — the core repeated entry
- **Ketone value (mmol/L)** — only when prompted (CBG >12, or T1 fasted/unwell)
- **Hypo signs present?** flag — critical because stroke/neuro patients mask symptoms (JBDS §8.2, §12)
### Group D — Event-driven (one tap when something changes)
- **Feed state change:** running / paused (planned) / **stopped (unexpected)** / restarted — + time + **reason** (medication, physio/procedure, shower, vomiting, tube blocked/displaced, refused) — these are the documented hypo triggers in JBDS Table 2
- **Insulin administered:** which dose + units + time + feed-related? (feeds the IOB model)
- **Feed-related dose withheld** — auto-prompted by the tool when feed is stopped
- **Hypo treatment given:** what + time
- **Carbohydrate/feed-rate change** (during feed titration)
